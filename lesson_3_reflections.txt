Q) When would you want to use a remote repository rather than keeping all your work local?

A) You want to use a remote repository when collaborating with others or being able to upload work from
another computer.


Q) Why might you want to always pull changes manually rather than having Git automatically stay 
up-to-date with your remote repository?

A) You might want to manually pull to control the file you are working on without interference from updates.
Updates can also bring new bugs.


Q) Describe the differences between forks, clones, and branches. When would you use one instead of another?

A) Fork is used to clone a git repository into your own git repository. Clone is used to clone your git repository
into your own local computer. You can also use clone to clone a local repository into another location in your 
local computer. Branches are used edit the master to keep the master intact and functioning and giving freedom
to explore in the branch.
